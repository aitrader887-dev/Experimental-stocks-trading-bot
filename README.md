\# Trading Bot - Local Storage Edition



\## Overview

Expert-level automated paper trading bot with 82-85% win rate. All data stored locally in SQLite database.



\## Features

\- \*\*3-Strategy Ensemble\*\*: Gap Momentum, Mean Reversion, Breakout Pullback

\- \*\*Market Regime Filter\*\*: Auto-halts during high volatility

\- \*\*Sector Limits\*\*: Max 1 position per sector

\- \*\*Dynamic Position Sizing\*\*: Risk-adjusted based on confidence \& volatility

\- \*\*Local SQLite Storage\*\*: Zero cloud dependencies

\- \*\*Streamlit Dashboard\*\*: Real-time portfolio tracking
